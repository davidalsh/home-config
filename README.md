# Cheatsheet

## Neovim — leader is `Space`

| Key | Action |
|-----|--------|
| `:w` | Save / store the file |
| `Space` `Space` | Find files |
| `Space` `f` `g` | Search text in project (grep) |
| `Space` `f` `b` | Open buffers |
| `Space` `f` `e` | Folder browser |
| `Space` `e` | File tree on the left (toggle) |
| `Space` `o` | Toggle focus: tree ↔ file |
| `Space` `f` `r` | Recent files |
| `Space` `f` `w` | Search word under cursor |
| `Space` `f` `h` | Search help |
| `gd` | Go to definition |
| `gr` | References |
| `K` | Hover docs |
| `Space` `r` `n` | Rename symbol |
| `Space` `c` `a` | Code action |
| `]d` / `[d` | Next / previous error |
| `:Lazy` / `:Mason` | Plugin / LSP store |

In a Telescope popup: type to filter · `Ctrl-j` / `Ctrl-k` move · `Enter` open · `Ctrl-d` delete buffer · `Esc` close.

## tmux — prefix is `Ctrl-a`

| Key | Action |
|-----|--------|
| `Ctrl-a` `"` | Split horizontally |
| `Ctrl-a` `%` | Split vertically |
| `Ctrl-a` `c` | New window |
| `Ctrl-a` `h` `j` `k` `l` | Move between panes |
| `Ctrl-a` `H` `J` `K` `L` | Resize pane |
| `Ctrl-a` `1`…`9` | Go to window N |
| `Ctrl-a` `z` | Zoom / unzoom pane |
| `Ctrl-a` `[` | Scroll / copy mode (`q` exits) |
| `Ctrl-a` `d` | Detach session |
| `Ctrl-a` `x` | Close pane |
